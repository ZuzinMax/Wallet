*** Приложение "Личный финансовый кошелек" с последовательным иерархическим меню через интерфейс консоли. ***

* Функционал приложения:
	- Добавление, изменение и удаление записей о личных финансовых операциях с автоматическим расчётом баланса кошелька
	- Просмотр полного списка операций
	- Поиск по записям (Категория, Дата, Сумма)
	- Просмотр текущего баланса
	- Очистка списка операций с последующим обнулением баланса

Для выбора действия используются цифры от 0 до 9, где 0 - Выход

Интерфейс на русском языке и интуитивно понятен.
